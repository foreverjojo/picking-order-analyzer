# 📦 撿貨單分析系統

艾薇手工坊 - 自動化生產統計工具

## 功能

- 📄 支援 MOMO（含 PDF 偵測）、官網、蝦皮、橘點子 撿貨單
- 🤖 新功能：若上傳的 PDF 在商店欄位包含 "MO店"（範例：艾薇手工坊MO店），系統會自動辨識為 MOMO 撿貨單，並套用 MOMO 的映射規則，免手動選擇
- 🔄 智慧商品名稱映射
- 📊 自動填入報表範本
- 📥 保留完整 Excel 格式輸出

## 使用方式

1. 上傳撿貨單文件（Excel 或 PDF）
2. 確認商品映射對應
3. 上傳報表範本
4. 下載填入數據的報表

## 支援格式

| 平台 | 格式 |
|------|------|
| MOMO | Excel (.xlsx), PDF (.pdf)（若 PDF 含 "MO店" 則會自動辨識為 MOMO） |
| 官網 | Excel (.xlsx) |
| 蝦皮 | PDF |
| 橘點子 | Excel (.xlsx) |
