# 📦 撿貨單分析系統

艾薇手工坊 - 自動化生產統計工具

## 功能

- 📄 支援 MOMO、官網、蝦皮、橘點子 撿貨單
- 🔄 智慧商品名稱映射
- 📊 自動填入報表範本
- 📥 保留完整 Excel 格式輸出

## 使用方式

1. 上傳撿貨單文件（Excel 或 PDF）
2. 確認商品映射對應
3. 上傳報表範本
4. 下載填入數據的報表

## 支援格式

| 平台 | 格式 |
|------|------|
| MOMO | Excel (.xlsx) |
| 官網 | Excel (.xlsx) |
| 蝦皮 | PDF |
| 橘點子 | Excel (.xlsx) |
